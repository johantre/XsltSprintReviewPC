CONTEXT

Load Voice Maps: Extend GDF schema & tools for phonemes

  • Names model has been extended with Phonemes. (TTOM-3_16-1_0-SNAPSHOT)
Problems and Solutions:
  • Extended AllCodes generator for Phoneme feature and attributes b/c the extension of the model with a Phoneme as a specialization of Name wasn't supported.
  • Introduced name type for GDF name records to differentiate between Name feature and Phoneme feature.
  • Added functionality to add sub attributes to existing composite attributes. In a post process the Phoneme feature / attributes will be created and added to existing composite attributes e.g. Name Transliteration Set.
  • Adapt Name Normalization to take attributes into account. Phonemes could have the same Phoneme text but different additional attributes. This results in a conflict with the Name Normalization.

STORY
