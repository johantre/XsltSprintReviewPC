RMC: make NCS22 compliant.

Goals:
  • The changes on TIF22 should not cause a crash on the RMC run.
Impact:
  • Currently the RMC is only using a limited set of features and attributes.
  • TIF22 has no impact on them, except for the new language codes.
Solution:
  • The TTOM model must be extended with the new language codes.
